## Project Description
This week focuses on learning and applying DOM (Document Object Model) manipulation using JavaScript. Two mini-projects were built:

1. DOM Practice – exploring and accessing elements  
2. Interactive To-Do List – building a functional task manager  

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## Project 1: DOM Practice

### Description
This project demonstrates how to access and manipulate HTML elements using JavaScript.

### Features
- Select elements using:
  - getElementById
  - querySelector
  - getElementsByClassName
- Navigate DOM:
  - Parent, children, siblings  
- Log elements to the console  
- Practice DOM traversal  

### Lessons Learned
- Understanding DOM structure  
- Selecting and modifying elements  
- Debugging JavaScript errors  

---

## Project 2: Interactive To-Do List

### Description
A fully functional to-do list that allows users to manage tasks dynamically.

### Features
- Add new tasks  
- Mark tasks as completed using a checkbox ✅  
- Delete tasks ❌  
- Styled UI with borders, spacing, and background  
- Real-time updates
